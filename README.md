# Python First Calculator

## Description
This is a menu-driven calculator built using Python. It performs basic arithmetic operations and is designed to help beginners understand Python programming concepts such as functions, loops, conditional statements, and user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous execution until the user exits

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository.
2. Open the project in VS Code or any Python IDE.
3. Run the Python file.
